## Как работает игра **Crazy Time**? 🎮

### 1. **Основное колесо** 🎡

В центре игры — большое колесо с 54 секторами, которые содержат различные числа (1, 2, 5, 10) и бонусные раунды, такие как **Cash Hunt**, **Pachinko**, **Coin Flip** и **Crazy Time**. Игроки делают ставки на один из этих секторов, и затем ведущий крутит колесо, чтобы определить результат.

- **Числовые сектора** (1, 2, 5, 10) — это обычные сектора, на которых выплачиваются выигрыши, в зависимости от ставки.
- **Бонусные сектора** — если колесо останавливается на одном из них, запускается дополнительный бонусный раунд, который может значительно увеличить ваши выигрыши.

### 2. **Бонусные раунды** 🎁

- **Cash Hunt** — на экране появляется сетка с мишенями, каждая из которых скрывает определенную сумму. Игрок выбирает мишень, и его выигрыш определяется случайным образом.
- **Pachinko** — на экране появляется доска с пинами, и шарик падает, определяя ваш выигрыш в одном из секторов с множителями.
- **Coin Flip** — игроку предстоит выбрать одну из двух сторон монеты, чтобы узнать, какой множитель он получит.
- **Crazy Time** — самый крупный бонус, где игроки могут выбрать одно из трёх колес, на которых скрыты множители, умножающие выигрыш до невероятных сумм.
